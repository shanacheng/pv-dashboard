# pv-dashboard


## Data Sources
The dataset used is data merged from the following sources: 
1. https://mappingpoliceviolence.org/
2. https://www.pewforum.org/religious-landscape-study/compare/party-affiliation/by/state/
3. https://data.census.gov/
4. https://github.com/washingtonpost/data-police-shootings
5. https://www.kaggle.com/stevechadwick/police-killings-analysis/data?select=police_killings.csv
